# UI设计模板 - 深思造物平台

## 整体布局
```
┌─────────────────────────────────────────────────────────────────────┐
│ 顶部导航栏 (面包屑导航 + 用户信息)                                   │
├───────────────┬─────────────────────────────────────────────────────┤
│ 左侧导航栏     │ 主内容区                                           │
│               ├─────────────────────────────────────────────────────┤
│               │ 页面标题 + 操作按钮                                 │
│               ├─────────────────────────────────────────────────────┤
│               │ 筛选/搜索区域                                       │
│               ├─────────────────────────────────────────────────────┤
│               │ 数据表格                                           │
│               ├─────────────────────────────────────────────────────┤
│               │ 分页组件                                           │
│               ├─────────────────────────────────────────────────────┤
│               │ 底部版权信息                                       │
└───────────────┴─────────────────────────────────────────────────────┘
```

## 左侧导航栏

### 结构
```
- 主导航项 (可折叠)
  - 子导航项
  - 子导航项
- 主导航项
  - 子导航项
```

### 样式
- 宽度：固定约200px
- 背景色：#ffffff
- 边框：右侧1px solid #ebeef5
- 选中项：背景色#ecf5ff，文字色#409eff，左侧边框4px solid #409eff
- 未选中项：文字色#606266
- 图标：与文字颜色一致，位于左侧

## 主内容区

### 1. 顶部区域
- 面包屑导航：字体大小14px，颜色#909399，分隔符"/"
- 页面标题：字体大小18px，颜色#303133，字体权重500
- 操作按钮组：位于标题右侧，包含新增、删除等按钮

### 2. 筛选/搜索区域
#### 布局
- 水平排列，左侧输入框/下拉框，右侧操作按钮
- 元素间距：16px

#### 表单控件样式
- **输入框**：
  - 宽度：根据需求设置（如200px-300px）
  - 高度：32px
  - 边框：1px solid #dcdfe6
  - 圆角：4px
  - 占位符：颜色#c0c4cc
  - 聚焦状态：边框颜色#409eff，阴影0 0 0 2px rgba(64, 158, 255, 0.2)

- **下拉选择框**：
  - 宽度：根据需求设置（如150px-200px）
  - 高度：32px
  - 边框：1px solid #dcdfe6
  - 圆角：4px
  - 下拉图标：位于右侧

- **日期选择器**：
  - 宽度：根据需求设置（如150px-200px）
  - 高度：32px
  - 边框：1px solid #dcdfe6
  - 圆角：4px
  - 日期图标：位于右侧

- **按钮**：
  - 主按钮：蓝色背景(#409eff)，白色文字，高度32px，圆角4px
  - 次要按钮：白色背景，灰色边框(#dcdfe6)，灰色文字(#606266)，高度32px，圆角4px
  - 按钮文字：字体大小14px，字体权重500

### 3. 数据表格
#### 基本样式
- 宽度：100%
- 背景色：#ffffff
- 边框：1px solid #ebeef5
- 行高：48px
- 表头：背景色#f5f7fa，文字色#909399，字体大小14px，字体权重500
- 表体：
  - 奇数行背景色#ffffff
  - 偶数行背景色#fafafa
  - 鼠标悬停：背景色#ecf5ff

#### 列类型
1. **复选框列**：
   - 宽度：50px
   - 居中显示

2. **序号列**：
   - 宽度：60px
   - 居中显示
   - 文字色#909399

3. **文本列**：
   - 根据内容设置最小宽度
   - 左对齐
   - 文字色#303133

4. **状态列**：
   - 宽度：100px
   - 居中显示
   - 使用标签样式：
     - 启用：绿色标签，文字"启用"
     - 禁用：橙色标签，文字"禁用"
     - 审核中：橙色标签，文字"审核中"
     - 通过：绿色标签，文字"通过"
     - 驳回：红色标签，文字"驳回"

5. **日期时间列**：
   - 宽度：180px
   - 居中显示
   - 文字色#909399
   - 格式：YYYY-MM-DD HH:mm:ss

6. **操作列**：
   - 宽度：150-200px
   - 居中显示
   - 操作按钮：
     - 查看：蓝色文字按钮
     - 编辑：蓝色文字按钮
     - 删除：红色文字按钮
     - 禁用/启用：灰色文字按钮
     - 更多操作：三点图标

### 4. 分页组件
- 位置：表格下方，右侧对齐
- 元素：
  - 总数显示："共X条"
  - 每页条数选择：下拉框(10/20/50/100条)
  - 页码导航：上一页、页码、下一页
  - 跳转输入框："前往X页"
- 样式：
  - 文字色#606266
  - 当前页码：蓝色背景(#409eff)，白色文字
  - 其他页码：白色背景，灰色边框

## 色彩系统

### 主色调
- 蓝色：#409eff（主要操作、链接、选中状态）

### 辅助色

- 橙色：#e6a23c（禁用状态、警告提示、审核中）
- 红色：#f56c6c（删除操作、错误提示）

### 中性色
- 主要文字：#303133
- 次要文字：#606266
- 辅助文字：#909399
- 边框颜色：#ebeef5
- 背景色：#f5f7fa
- 白色：#ffffff

## 字体系统
- 主要字体：无衬线字体（如微软雅黑、Helvetica）
- 标题：18px，字体权重500
- 正文：14px
- 辅助文字：13px
- 表格数据：14px

## 组件库
- 使用Element Plus组件库
- 主要组件：
  - 导航菜单：el-menu
  - 面包屑：el-breadcrumb
  - 按钮：el-button
  - 输入框：el-input
  - 下拉选择器：el-select
  - 日期选择器：el-date-picker
  - 表格：el-table
  - 分页：el-pagination
  - 标签：el-tag
  - 图标：@element-plus/icons-vue

## 示例页面生成指令

### 使用方式
提供页面需求时，请包含以下信息：
1. 页面名称
2. 左侧导航结构
3. 筛选条件（字段名、类型、默认值等）
4. 表格列信息（字段名、类型、宽度、操作等）
5. 操作按钮（新增、删除等）

### 示例指令
```
生成页面：系统设置-参数字典管理
左侧导航：系统设置 > 参数字典
筛选条件：
- 参数字典名称：输入框
- 应用类型：下拉选择（B公司、K公司、通用）
- 状态：下拉选择（全部、启用、禁用）
- 创建时间：日期范围选择器
表格列：
- 序号：序号列
- 参数字典名称：文本列
- 版本号：文本列
- 应用类型：文本列
- 更新内容：文本列
- 状态：状态列（启用/禁用）
- 创建人：文本列
- 创建时间：日期时间列
- 更新时间：日期时间列
- 操作：操作列（查看、编辑、删除）
操作按钮：新增、删除
```

## 注意事项
1. 所有页面保持一致的设计风格和交互模式
2. 响应式设计：适配不同屏幕尺寸
3. 无障碍设计：遵循WCAG标准
4. 性能优化：表格数据分页加载，避免一次性加载大量数据
5. 用户体验：提供清晰的操作反馈，如加载状态、成功/失败提示
