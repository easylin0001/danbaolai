会员电商系统中控台 - DANBAOLAI文档

一、功能介绍

为便于管理员及时对直播间内容进行管控，并能迅速实现直播间商品的上架与下架操作，特增设中控台功能。通过该中控台，可迅速全面掌握整个直播间的所有信息。

二、操作步骤

三、页面介绍

1、开始直播

未开播状态下，管理员可进入直播中控台，查看橱窗内设置的商品。
点击“开始直播”后，直播间即转变为“直播中”状态，此时移动端便可看到直播画面。

2、直播中

直播画面：实时查看直播画面，确保管理员实时掌握直播流程。
结束直播：可点击结束直播，直播间即变为“已结束”状态，用户端提示直播已结束
直播数据：实时显示直播间成交、观看数据。

成交金额：用户进入直播间下单并购买的实付金额之和；去除退款金额
实时在线人数：当下观看直播间的人数
商品点击-成交率：直播间成交人数/直播间商品点击人数
累计观看人数：进入直播间的人数，同一用户去除重复进入的次数
成交人数：直播间内下单的用户数
成交件数：直播间内购买的商品数量

用户评论：直播间用户评论实时滚动，可将鼠标悬浮至用户评论上，回复、删除评论；也支持禁言对应用户
快捷回复：预先在“店铺->客服管理->客服话术”中维护直播话术，可在中控台页面通过模糊搜索话术，以实现快捷回复。
观众列表：查看直播间所有观众的观看时长、购买商品数、订单金额等。支持设置直播虚拟人数
商品：在直播间内进行商品的上架与下架操作，下架后的商品，可在待播商品列表中查看，且支持重新上架；同时，支持通过拖动的方式调整商品顺序，也可通过修改商品序号快速变更顺序；在中控台内，能够快速修改直播商品的直播库存以及虚拟销量；支持编辑商品卖点，该卖点将在移动端商品列表弹窗中展示；还支持实时查看每个商品点击量、成交金额、成交数量等销售数据。
讲解：点击后，移动端将该商品弹出至直播页面，并在直播商品列表置顶
